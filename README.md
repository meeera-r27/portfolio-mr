# 🌐 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website developed to showcase my skills, projects, internship experience, education, and technical expertise as an aspiring MERN Stack Developer.

The portfolio features a clean dark-themed interface, smooth animations, responsive layouts, and an intuitive user experience.

---

## 🚀 Features

- Modern and responsive design
- Dark-themed professional UI
- Smooth scrolling navigation
- Animated hero section
- Typing terminal animation
- Scroll reveal effects
- Animated statistics counter
- About Me section
- Skills showcase
- Featured projects
- Internship experience
- Education & certifications
- Contact section
- Mobile-friendly navigation
